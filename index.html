明星带你去旅行
活动名称 : 明星带你去旅行
活动时间 : 永久
活动范畴 : 大神游戏平台所有游戏
活动介绍 : 对于追星族来说，为爱豆花钱是种快乐；对于程序员来说，今晚不用加班是种快乐；对于父母来说，熊孩子终于开学了是种快乐；而对于 游戏玩家来说，快乐是什么？是打BOSS、爆头时那一瞬间的快感？是经历几个月的日日夜夜终于升级熬成了大神？是打怪完成后终于爆出 了超好的装备？还是出了新副本又可以崭露头角？以上的一切，皆是看不到摸不着的“快乐”，带给你的感受消纵即逝，今天，大神游戏将 重新定义快乐，带给你真实的荣耀体验！
大神游戏现与各企业合作（海信、乐奔租车、雨龙租车、iser饮料等企业），让您在玩游戏的同时，享受生活，快乐生活。您在玩游戏时产 生的每一笔消费，都会有回报。当您在游戏内消费达到一定金额，大神游戏可以让明星们带您去境外旅行，明星您来挑，主持人您来定。
参与方式 : （1）您在大神游戏平台里选择任意一款游戏参与“人人有礼”活动一次性充值12000元的玩家，即可获得“明星带你去旅行”境外游7天机会。
（2）您在大神游戏平台里选择任意一款游戏累计充值达到24000元，即可获得“明星带你去旅行”境外旅游7天机会（玩家可携带一个人）。
（3）参与大神游戏俱乐部活动“海信大神快乐宝箱”活动，也能获得“明星带你去旅行”境外旅游7天。

活动名称 : 海信大神快乐宝箱
活动时间 : 永久
活动范畴 : 大神游戏平台所有游戏
活动介绍 : 玩家在大神游戏平台中选择任意一款游戏参与并充值后，即可获得参与活动的资格。活动通过开启宝箱和自主翻牌的模式，抽取不同的礼物，累 积相应的积分，最终获得参与境外旅游的资格。在活动里，是没有“谢谢参与”这一选项的，玩家在活动过程中充值的每一笔费用，都会得到相 应的现实礼物回报，从而改善玩家的生活质量，改变玩家的生活方式，真正达到大神游戏想做到的游戏改变生活的目的。
活动规则 : 开启宝箱会消耗累计充值金额，“初入大神”，“创神纪元”，“炼神之路”，“神王之光”“大神戎装”五种宝箱中，每个宝箱藏有9张黄金卡 牌，4张现实礼物，5张积分，随机抽取一样，最后一个宝箱“神王戎装”宝箱中装着“明星带你去旅行”和2个神秘礼物。
